Tutorial de test unitarios en AngularJS

El tutorial revisa los conceptos básicos para hacer test unitarios en AngularJS con Jasmine y Karma. El proyecto esta basado en js-seed. Puedes referirte a este proyecto para saber como usarlo y configurarlo.

Preparación

Para ejecutar los test del tutorial, debes tener instalado en tu sistema los siguientes componentes:

  • el controlador de versiones GIT
  • la plataforma NodeJS
  • el sistema de buildeo Gulp

Debes instalar Gulp cuando tengas NodeJS instalado y funcionando. Es necesario instalarlo de forma global invocando en la linea de comandos:

npm install -g gulp

Una vez instalados los prerequisitos, clona el repositorio angular-testing-tutorial e instala las dependencias.

git clone https://github.com/jseto/angular-testing-tutorial.git
npm install
